# guess-game

Угадай предмет/животное.

**Правила**

- Игрок задаёт до 10 вопросов.
- На каждый вопрос игра отвечает строго: `да`, `нет` или `не знаю`.
- Игрок может попробовать угадать, задав вопрос вида: «Это …?».

## Сделать игру доступной по ссылке (Render)

Цель: чтобы пользователю нужно было просто открыть URL, и игра работала без твоего компьютера.
